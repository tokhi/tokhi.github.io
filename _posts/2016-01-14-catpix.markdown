Install the gem:
{% highlight java}
gem install catpix
{% endhighlight %}

Very simple to do:

{% highlight java %}
require 'catpix'
Catpix::print_image "caveman1.png",
  :limit_x => 1.0,
  :limit_y => 0,
  :center_x => true,
  :center_y => true,
  :bg => "white",
  :bg_fill => true
{% endhighlight %}
